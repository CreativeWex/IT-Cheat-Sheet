
# Операции с таблицами
Создание таблицы
```
CREATE TABLE название
(
    Имя поля тип(макс кол-во символов для ввода) [ограничения],
    Имя поля тип(макс кол-во символов для ввода) [ограничения],,
    ...
    Имя поля тип(макс кол-во символов для ввода) [ограничения],,
    [ограничения],
    [первичный ключ],
    [внешний ключ]
);
 ```
PRIMARY KEY - обязательное поле

<br>

`\dt` - Список таблиц

`\d НАЗВАНИЕ` - информация о таблице
